# Frontend Notes

This folder contains the React + Vite frontend for PharmaChain.

For the full local setup and run instructions, use the repository root guide:

- See `../README.md`

Important runtime detail:

- `src/config.json` is generated or updated by `blockchain/scripts/deploy.js`
- Do not treat `src/config.json` as a static hand-maintained file during local development

Useful frontend-only commands:

```bash
pnpm run dev
pnpm run lint
pnpm run build
```
